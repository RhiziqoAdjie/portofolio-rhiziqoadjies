Publish Perubahan

Penting: website ini adalah situs statis tanpa server/database. Semua perubahan di dashboard ini tersimpan di penyimpanan browser (localStorage) perangkat yang kamu pakai sekarang — jadi hanya kamu yang bisa melihatnya lewat browser ini, dan pengunjung lain belum melihat perubahan tersebut. Untuk membuat perubahan tampil permanen bagi semua orang, unduh file data.js terbaru lalu ganti file lama di server/hosting kamu.
  1. Klik "Lihat Preview" untuk mengecek tampilan sebelum dipublikasikan.
  2. Klik "Unduh data.js" — file ini berisi semua konten terbaru (Experience, Projects, Certificate, Skills, teks ID/EN).
  3. Ganti file assets/js/data.js di folder website/hosting kamu dengan file yang baru diunduh.
  4. Upload ulang ke hosting (atau commit & push jika pakai GitHub Pages/Netlify/Vercel).

Ganti Password Admin

Password hanya tersimpan di browser ini (bukan sistem login server). Gunakan sekadar sebagai penghalang ringan agar orang lain tidak iseng mengubah isi website.
